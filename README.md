# 💰 Monetra — Finance Tracker

Monetra is a modern web-based finance tracker built to help users manage income, expenses, and financial goals with clarity and control.

Designed with simplicity and performance in mind, Monetra provides a clean dashboard experience powered by Laravel, Tailwind CSS, MySQL, and Docker.

---

## 🚀 Features

- 📊 Track income and expenses
- 📈 Financial insights & analytics *(coming soon)*
- 🎯 Goal management
- 🌙 Dark mode support
- ⚡ Fast and responsive UI
- 🐳 Dockerized development environment

---

## 🛠️ Tech Stack

- **Fullstack Framework:** Next.js 15
- **UI & Styling:** Tailwind CSS 4
- **Database:** MySQL 8
- **ORM:** Prisma
- **Authentication:** NextAuth.js / Auth.js *(Optional)*
- **Data Visualization:** Recharts
- **Environment & Container:** Docker
